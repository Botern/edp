通过命令行使用

安装完成后，可以通过命令行下运行edp命令使用。常见的命令语法是：edp 子命令 参数。

查看当前支持的命令列表

初始化安装只会安装很少量最基础的命令，以节省安装的下载时间成本。其他命令（比如查询现有package的search命令）将在运行时自动安装。看看edp内置哪些命令
