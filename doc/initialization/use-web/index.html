通过web使用

除了传统的通过命令行方式使用之外，EDP基于以下考虑，提供了Web的使用方式。

  1. 为命令行使用不熟练的同学提供便利的使用方式。
  2. 能在运行命令前清晰地看到命令支持的参数和完整的帮助信息。
  3. 临时在开发机上调试和修改时，可以无需登录开发机。
